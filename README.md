# YT Downloader

A standalone Windows app to download YouTube videos and playlists as **MP3**, **WAV**, **MP4**, or **WebM**. Paste a URL, pick a format, click Download.

![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

## Download

**[→ Get the latest release](https://github.com/YOUR_USERNAME/Youtube_DownloadManager/releases)** — Download `YTDownloader.exe`, run it, done. No install needed.

## How to Use

1. Run `YTDownloader.exe`
2. Paste a YouTube URL or playlist URL
3. Choose output format (MP3, WAV, MP4, WebM)
4. Pick an output folder (default: Downloads)
5. Click **DOWNLOAD**

Files are saved to your chosen folder.

## Features

- **Standalone** — No installation, works on any Windows PC
- **Video & audio** — MP3, WAV, MP4, WebM
- **Playlists** — Download entire playlists in one go
